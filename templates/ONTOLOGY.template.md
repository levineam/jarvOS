## How Your Assistant Uses This

**Goal tracing:** Every project should trace back to a Goal. Goals trace to Mission. If a project doesn't connect to anything, it's an orphan — flag it.

**Alignment checks:** During reflection, compare recent work against Mission and Values. If you're spending time on things that don't connect, surface the drift.

**Orphan detection:** Projects not linked to any Goal. Goals not linked to any active project. Predictions that haven't been reviewed in 90 days.

**Belief updates:** When evidence contradicts a Belief or a Prediction is proved wrong, route the update here. Beliefs and Predictions are living documents, not monuments.

**Prioritization:** When multiple things compete for attention, the hierarchy is: Mission > Values > Goals > Projects. Work that serves a Goal linked to Mission gets priority over orphaned work.
