# jarvOS v0.6.1

## Summary

v0.6.1 is a small patch release after v0.6.0. It fixes unintended `jarVOS`
capitalization in the public secondbrain docs/tests so the canonical product
spelling remains `jarvOS`.

## What's Included

- Capitalization-only cleanup in `modules/jarvos-secondbrain`.

## Fixes

- Replaced lowercase-j uppercase-V `jarVOS` strings with canonical `jarvOS` in
  public secondbrain surfaces.

## Breaking Changes

- None.

## Known Limitations

- Legacy `JarVOS` fixture names and historical docs remain for a separate,
  broader naming cleanup decision.
