# jarvOS v0.1.0

## Summary

First public preview of jarvOS, for people who want a portable AI operating-system layer they can run on top of OpenClaw or Hermes Agent.

This release is intentionally git-based. Clone the repo, run the smoke test, and adapt the markdown/runtime templates to your workspace.

## What's Included

- Portable core files for identity, persona, behavioral rules, and setup templates.
- Runtime setup paths for Hermes Agent and OpenClaw.
- Claude Desktop manual MCP hydration through the `boot jarvOS` prompt.
- Bootstrap installer smoke coverage for shipped files and idempotent setup.
- GBrain-first structured recall integration via `@jarvos/gbrain`: curated import planning, GBrain sync/embed wrappers, graph recall, runtime recall bundles, and public eval fixtures.
- Public release process documentation, GitHub Release template, and executable release readiness check.

## Fixes

- No patch-level fixes yet. This is the first public preview release candidate.

## Breaking Changes

- None known.

## Known Limitations

- No npm package is published for this release.
- Setup expects comfort with local developer tools and runtime-specific configuration.
- Claude Desktop startup auto-hydration is not included; Desktop uses the manual MCP hydration flow.
- Andrew's live private workspace is not included in the public baseline.
- The public repo includes GBrain integration code and fixtures, not GBrain itself, private generated pages, private manifests, or a prebuilt local graph.
- The public repo is a portable baseline, not a zero-config clone of the live jarvOS system.
