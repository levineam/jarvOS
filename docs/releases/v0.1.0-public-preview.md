# jarvOS v0.1.0 Public Preview

`v0.1.0` is the first public preview release of jarvOS.

## Release Status

- Status: candidate intake open
- Paperclip release lane: `SUP-1648`
- GitHub Release: not published yet
- Final approval: pending

## Included Surface

- Portable core behavior files in `core/`
- User-fillable templates in `templates/`
- Starter kit project and governance templates
- Runtime adapters for OpenClaw, Hermes, Codex, and Claude
- Public modules for memory, ontology, secondbrain, GBrain, runtime kit, and
  agent context
- Smoke tests for clean-checkout structure and bootstrap behavior

## Release Notes Draft

### Highlights

- Ships jarvOS as a plain git repo that can be cloned, inspected, forked, and
  adapted without a hosted service.
- Separates portable agent behavior from runtime-specific setup.
- Includes public module code for memory, ontology, secondbrain, GBrain, runtime
  adapter, and agent-context workflows.
- Provides smoke tests so users can verify a clone before adapting it.

### Known Boundaries

- This is a public baseline, not Andrew's private live workspace.
- Filled-in memory, ontology, vault content, Paperclip IDs, runtime secrets, and
  machine-local paths stay private.
- Pre-`v1.0`, some behavior and module APIs may change as the public baseline is
  tested by real users.

### Verification Before Publish

- `bash scripts/smoke-test.sh`
- `npm test`
- GitHub Actions CI on the final release PR or tag
